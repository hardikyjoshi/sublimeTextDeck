Understanding the interface

Useful Shortcuts

  • Open last closed file: ctrl (cmd) + shift + t
  • Select file using number: alt (cmd) + (0-9)
  • Open any file by name in project: ctrl(cmd) + p
  • Adding comments: ctrl + / or ctrl + shift + /
  • Find and Replace
  • Multiple Selections
    • ctrl (cmd) + click
    • Select lines(ctrl(cmd) + shift + l and then alt + direction keys)
    • Select word -> Quick Find All -> Use alt and direction keys to replace
    • Replacing Paragraphs: Select word -> Quick Find All -> Select Content ctrl + shift + a (shift + cmd + a) -> Copy Content in a new file, make changes -> Select lines ctrl(cmd) + shift + l -> Paste in original file
